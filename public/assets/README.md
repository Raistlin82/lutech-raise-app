# Logo Assets

This directory contains the Lutech logo assets used in the RAISE App.

## Required Files

Place the following logo files in this directory:

- `lutech-logo.png` - Main Lutech logo (recommended: 200x50px or similar aspect ratio)
- `lutech-logo-sm.png` - Small version for compact displays (optional, recommended: 100x25px)

## Usage

The logo is displayed in:
- Sidebar header (with author credit "by Gabriele Rendina")
- Footer component (with copyright notice)

## Fallback

Until the actual logo files are provided, the application will display a text-based logo using the "Lutech" brand name.
